# 用药提醒应用

一个帮助用户管理药品信息和按时服药的健康管理应用。

## 功能特点

- 药品管理：添加、编辑、删除药品信息
- 服药提醒：按时段（早餐、中餐、晚餐）发送提醒
- 用药记录：记录每日服药情况
- 数据统计：展示七日内服药完成率

## 技术栈

- 数据存储：本地数据库（SQLite）
- 用户界面：响应式 Web 技术
- 通知系统：系统原生通知
